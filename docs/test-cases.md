# テストケース: 食品流通システム 食品受発注DXシステム

## テスト概要

### テスト目的

本テストケースは、食品業界向け受発注DXシステム「食品流通システム」の品質を保証するために策定された包括的テストケース集である。受注（OCR読取）から発注確認、在庫管理、出荷処理、請求書発行、返品処理までの全業務フローを網羅し、各機能の正常動作およびエラーハンドリングを検証する。

### 対象範囲

- トップページ（ランディングページ）およびナビゲーション
- ワークフローステップバー（WorkflowStepBar）
- AI-OCR 伝票読取機能（`/ocr`）
- 発注管理機能（`/orders`）
- 在庫管理機能（`/inventory`、`/inventory/history`）
- 会計連携機能（`/accounting`、`/accounting/invoices`、`/accounting/export`）
- 返品管理機能（`/returns`、`/returns/new`）
- 商品マスタ（`/products`）
- 各種 API エンドポイント（`/api/orders`、`/api/ocr`、`/api/inventory`、`/api/invoices`、`/api/returns`）

### テスト環境

| 項目 | 内容 |
|------|------|
| システム | 食品流通システム |
| フレームワーク | Next.js (App Router) |
| データベース | Prisma + SQLite（開発環境） |
| ブラウザ | Chrome 最新版、Safari 最新版、Firefox 最新版 |
| 画面サイズ | デスクトップ（1920x1080）、タブレット（768x1024）、モバイル（375x812） |
| テストデータ | 高知アイス株式会社を想定した食品関連データ |

### テストデータ例

| データ種別 | 値 |
|-----------|-----|
| 会社名（発注元） | イオン高知店 |
| 会社名（仕入先） | 高知アイス株式会社 |
| 商品1 | 抹茶アイスクリーム（PRD-101）、¥450/個 |

| 請求書番号例 | INV-20260318-001 |
